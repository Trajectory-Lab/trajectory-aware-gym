# Results

Published experiment artifacts live in this directory.

- `summary.md` is auto-generated by `poe publish --update-summary-only`.
- The formal result tables only count the four production configs:
  `hotpotqa-tool`, `hotpotqa-notool`, `orz57k-tool`, and `orz57k-notool`.
- Production config directories hold protocol-compliant publishes used for
  formal comparison against the GEM baselines.
- `orz57k-protocol-drift/` archives published Orz57K/Qwen/Ollama variants that
  are intentionally excluded from the formal tables because the run config
  drifted from the canonical protocol, used a non-production config, or had
  run-health issues.
